# Generated Agent API Catalog

This directory is generated by the `admin-api:agent-docs-generate` Nx target.
Check it with `nx run admin-api:agent-docs-check`.

- `catalog.json`: full endpoint catalog copied as a reference fixture.
- `openapi.json`: OpenAPI-flavored REST description copied as a reference fixture.
- `stackos-tools.json`: compact REST tool manifest copied as a reference fixture.
- `schema-constraints-audit.json`: generated inventory of schema refinement rules
  extracted from Zod schemas.

Endpoint count: 326

The server remains the authority for auth, permission profiles, scoping, and `X-Acting-As-Account`.
Endpoint context metadata is authored with `@EndpointContext(...)` on the
controller method and only improves discovery and tool descriptions.

StackOS runtime inventory is fetched from the live Trackbooth server by the
manual `trackbooth.catalog.sync` action, not from these copied files.
